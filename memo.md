## 2. 生命のパターンを作る
・自己組織化  
生物は多様なパターンをつくる。設計図があるわけではなく、単純なピースで複雑な非線形な相互作用による自己組織化で作る。  

・チューリングモデル  
自然界の様々なパターンは化学反応と拡散過程という自己組織的なロジックで作られる。化学物質が一様に分布することにより不安定化し、自発的にいろいろなパターンが生み出されるというもの。  

・Gray-Scottモデル  
流入する抑制生のUと流出する興奮生のVが反応する。U+2V=3V  
さらにそれが拡散する。  
濃度に応じた拡散の項と化学反応の項と流入、流出の項で考える。
反応波として伝わっていく。  
流入量のパラメータfと流出量のパラメータkを調節することで様々なパターンをつくることができる。  

### セルラー・オートマトン  
細胞を意味するセルが周りのセルの状態により状態を更新していくモデル。よく知られている「ライフゲーム」もこれの一種。  
非線形なので、局所的に定義されたルールからはどんなパターンになるかは予測できないところが面白い。  
色々なバリエーションがあるが、共通のルールがあって、  
1,空間がある  
2,時間がある  
3,ゼルの状態がある  
4,セルの状態を変える条件がある  

・一次元セルラー・オートマトン  
一次元の配列で、隣二つの状態から自分の状態を決めるというルール。隣と自分で8通りあり、それに対し生きるか死ぬかで2の8乗=256通りのルールがある。  
実行すると見た目から４つに分けられる。  
クラス１：固定点  
クラス２：周期的
クラス３：カオス
クラス４：局在する構造を持つ複雑なパターン  
クラス４が面白くて自然界にある。貝殻の模様とからしい。  

・二次元セルラー・オートマトン（ライフゲーム）  
ライフゲームは二次元CAのルールセットの一つ。  
人口過剰、均衡状態、人口過疎、再生の４つのルールにしたがって動くモデル。  
初期値によってパターンが変わる。  
安定パターン、オシレータ、オグライダー、グライダーガンなどがある。  
初期値のデザインによって、計算機をシミュレートしたり、ライフゲーム自体をシミュレートしたりできるらしい。自己言及性を用いて、どんなノイズ下でも安定的に振る舞えるガッチスなどがある。  

生命の背後にはこのような自己組織化の仕組みがある。しかし、それらはノイズにとても弱く現実では使い物にならない。ハードの特製を上手く使うことで頑健性を持つことができるのかも。(ex.エイドリアン・トンプソンのFPGAと遺伝的アルゴリズムの実験)　　

## 3. 個と自己複製  
### 3.1 個の創発
自己の問題は難しいが、安定性と不安定性の間にあって、環境の変化に対応できるくらいの不安定性をもつシステムとして個が双発する。  
自らを生成し、維持できる最小単位の生命の機構をもつシステム「オートポイエーシス」を考える。  
・SCLモデル  
オートポイエーシス概念を公正的に理解するためのモデル。  
個の創発の問題に対し、自己の存在を決定するのはそれ自体の構成するプロセスによってである。という考え方を提示する。  
ランダムに移動する３つの分子と、その反応によって、細胞が自律的に膜を作る様子を再現する。これが個の形成と自己維持。  
分子：基質分子(S)、触媒分子(C)、膜分子(L)  
反応  
2S + C → L + C　（触媒によって基質分子が膜分子になる） L + L → L-L （膜分子は隣接すると結合して固定される）   
L → 2S　（膜分子は一定の確率で分解されて基質分子に戻る）  

・改良SCLモデル  
膜の自己生成を通した感覚運動系の起源を探る研究。
機能基質分子のみが膜を通れるようにし、機能基質分子によって触媒分子が動かされるようにする事で、自己維持のプロセスとともに運動が生じるようになった。これで、濃度を感知して運動する感覚運動カップリングが生じたと言えるのでは。  

・フォン・ノイマンの自己複製オートマトン  
自己複製を哲学だけではなく、科学で扱えるようにした。  
解釈をして製造する「マシン」と、コピーされる「テープ」（データ）に分けて考える事で、テープをコピーしてマシンで解釈することで複製を行うことができる。これはDNAの考え方。  
それをセルラー・オートマトン上にデザインした。  

・Golly  
ハッシュアルゴリズムを用いて高速化した自己複製オートマトンのシミュレーター。簡単に使えるらしい。  

その他にも様々な自己複製オートマトンがある。規模を小さくして限定的にすることで簡単にできるモデルもあるが、それは自明な自己複製でやり方がが決まってしまう。「非自明」な自己複製であることが大切で、そうでないと地球上の自己複製できるパターンは一つに決まってしまう。  

### 3.2 ノイズに強い自己複製  
今までの自己複製モデルはノイズに弱い。  
ノイズに強いものは膨大な計算量を使う、ガッチスのモデルくらい。それとフォンノイマンのモデルを組み合わせると作れるのかもしれない。  

・Gray-Scottモデルの化学反応系  
スポットが自己複製している様子がみられる。これを拡張して移動とかモデルようにしたモデルがある。  
この場合、テープのような設計図はなく、パラメータによって何が作られるか決まる。自己複製や進化、情報の継承を生命システムの創発として捉えるには、まだまだアイデアが足りない。  

・実機の自己複製機  
ペンローズの実機ユニット  
バネとフックを利用した木製おもちゃで、ぶつかる（ノイズ）ことによってフックで結合して、４つになると二つずつに分裂する。ゆらぐノイズの入った空間の自己複製。  

ナサニエルとクリサンタの実機ユニット  
エアホッケー台の上で回転するコウモリ型のユニットが走り回る。遺伝的アルゴリズムによってできた特異的な形で、ある決まった形にだけ合体するようになっている。  

ノイズに強くロバストにすると、結晶成長に近く、進化しないという問題がある。

### 3.3 自己複製と進化  
ロバストすぎると、指数関数的に増えて生態系で窒息してしまう、生態系としては多様性が大切。進化において、自己複製と多様性は相反する関係にある。生態系全体で議論することが必要になる。  

・「テープとマシン」モデル  
マシンは、テープのヘッドに一致する部分からテールに一致する部分までを「複製」して、遷移表にしたがって「翻訳」する。翻訳による「アクティブ変異」と外的なノイズによる「パッシブ変異」がある。  
翻訳によって違うテープが生まれる、自己の揺らぎもあるモデル。で、マシンとテープは数に比例して増えていき、古いマシンとテープは消滅していく。生態系世界をシミュレーションしていく。  

・コアネットワーク  
「テープとマシン」モデルにおいて外部ノイズがあまりないとシンプルなネットワークであるテープが対応するマシンを作ってそのマシンがそのテープを作り、他のテープがそれに対応するマシンを作ってという感じ。  
でも、ノイズを多くするとネットワークが複雑になり、AがBを作りBがAを作るというような他者依存的なネットワークを作るようになる。完全にプログラム的な書き換えによって維持されるネットワークを「コアネットワーク」と呼ぶ。  
自己を曖昧にして、環境変化に対するロバストネスを獲得し、全体として複製にいたるものが自然界にもあり、「準種」と呼ぶらしい。  

## 4. 生命としての群れ  
個体が生まれ、自己増幅ができるようになると、群れを成す。個体同士が強調しながら集団そのものの行動パターンや構造が創発するメカニズムが研究されてきた。  

### 4.1 創発現象  
創発とは、自律的に動く個体間の共同現象の結果、自己組織的にパターンや構造が現れる現象。  
フォンノイマンやチューリングらの反応拡散モデルも、ミクロな相互作用の結果、パターンが現れる創発現象。  
アリの群れが巣から餌のある場所までの最短経路を見つけ出す創発現象を模倣した計算方法。（アリコロニー最適化）  
魚や鳥の群れで、一匹が良い経路を発見すると群れの残りが素早くそれにならうことができる創発現象。（粒子郡最適化）  
Alifeで一番有名なのは、鳥のボイドモデル。CGの世界でとても画期的だった。  
MASSIVEという有名なソフトウェアがあり、ロード・オブ・ザ・リングのオークの群れによる大規模な戦闘シーンなど、映画やゲーム作品で広く使われている。  

### 4.2 ボイドモデル  
ボイドモデルでは、それぞれの個体が位置と向きを持っていて、  
「分離」(SEPARATION)：周りの個体とぶつからないように離れる  
「整列」(ALIGNMENT)：周りの個体と同じ方向に飛ぶように合わせる  
「結合」(COHESION)：周りの個体の中心方向へ向かうように向きを揃える  
の三つの力を使ってモデル化する。

これらのパラメータを変えることで群れの動きも変わって、  
1.swarm:ひとつに固まった群れを作る  
2.torus:集まって輪を作る  
3.dynamic parallel group:一つにまとまってダイナミックに動く  
4.highly parallel group:同じ方向をむいて整列し、直進的な動きをする  
などがある。  

これはただの運動学力学的なものだが、群れには、「郡知能」や「集団知」などがある。  
エサという「目的」を導入すると、動きがが変わるし、色々なバリエーションがあり得る。  

### 4.3 創発と生命の進化のプロセス  
生物の群れの創発現象は進化の賜物なはず。進化によって創発させたい。  
ボイドモデルにおいて数を増やしていくと動きが変わって、マクロな動きがミクロな動きに影響するようになる。  

個体内部に人工神経細胞モデルを持たせて、進化アルゴリズムを使って進化させた。餌場に到達できるかを適応度関数としたところ、群れを作る事でエサ場につきやすくなり、結果として群れを作る行動を進化させることができた。  
群れを作るようになると一気に多様性が生まれた。そのまま進化を続けても新しい行動が生まれることはなかった。  

次々と新たな創発現象が起きることを「多段階創発」と呼ぶが、それをそとから低高度地形を変更せずに起こすことは難しい。  

### 4.4 ネットワークとしての創発  
「隣接可能領域」（adjacent possible）という概念。  
実際に存在するモノから一ステップ離れた領域のことで、新しいモノが生まれる時には、隣接していた可能性のある領域が現実化されて、新しく隣接する領域が広がるのだ、という考え方。世界にはいつでも変化する可能性があるけど、一定範囲の変化のみが起こりうるということ。  
隣接可能領域は進化のプロセスとしての創発と捉えられる。  
隣接可能領域を作っていくものは何か？

一つはネットワークという集団。
集団になる事で生まれる知性がある。アリも、群れを作ることで合理的な判断ができるが、一方で、集団になることでお互いのホルモンを死ぬまで追いかけ続けるような非合理的な判断をすることもある。  

人の場合の一例はインターネット

・ウェブサービスの集団知
